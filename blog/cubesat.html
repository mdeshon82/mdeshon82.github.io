Why Build a CubeSat?

My goal is simple: a deployer-compatible 1U CubeSat that deploys an antenna, collects basic telemetry (temperature, GPS, attitude), and beacons it back on UHF/VHF amateur bands. Total budget under $2,000, built on weekends over 12–18 months.

I’m treating the satellite like a “space PLC” — modular subsystems orchestrated by a central controller with rigorous testing and fail-safes — exactly the same mindset I used when automating the honey bottling line.

Sourcing Everything on Amazon (Mostly)

No cleanrooms, no AS9100 vendors — just rugged, space-ish parts that overperform for the price. Here’s the core shopping list:

  • Structure: 6061 aluminum square tubing + 3D-printed endplates (~$50)
  • Power: 18650 cells, TP4056 chargers, 5V flexible solar panels, tiny MPPT (~$80)
  • OBC: Raspberry Pi Zero W or ESP32 (potted in epoxy for radiation tolerance) (~$20)
  • Comms: Si4463 UHF/VHF transceiver + nitinol deployable dipole (~$40)
  • Sensors: BMP280, MPU6050, small camera module (~$30)
  • Deployment: Spring-loaded hinges & nichrome burn-wire release (~$25)

Running total: ~$245 — the rest goes to DIY vacuum/thermal test gear.

Project Plan – From Breadboard to Orbit

Phase 1: Design & Prototyping (Months 1–3)

  • KiCad schematics
  • Breadboard OBC + radio
  • 3D-print mock frame
  • Milestone: “Hello world” beacon received on ground station

Phase 2: Assembly & Integration (Months 4–6)

  • Weld/bond aluminum frame
  • Solder power & sensor PCBs
  • MicroPython flight software with deployment logic
  • Build nitinol antenna canister + burn-wire release
  • Milestone: Fully stacked CubeSat running 24h stress test

Phase 3: Environmental Testing (Months 7–9)

  • Vibration table (modified subwoofer)
  • Thermal-vacuum chamber (pressure-cooker + fridge)
  • EMI and basic radiation hardening checks
  • Milestone: Survives full launch-to-LEO simulation

Phase 4: Certification & Launch (Months 10+)

Final Thoughts

This project is proof that space is becoming democratized. With Amazon parts, a PLC-style control philosophy, and relentless ground testing, a hobbyist really can build something that flies 500 km above Earth.
